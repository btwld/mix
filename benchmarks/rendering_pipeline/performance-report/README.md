Responsive web companion to `../PERFORMANCE_IMPACT_REPORT.md`. The page
visualizes the final release/AOT campaign, diagnostic pipeline split, state
transition contracts, accepted optimizations, rejected experiments, and
practical frame-budget interpretation.

```bash
npm install
npm run dev
npm test
```

The numerical source of truth remains the benchmark documentation and the
gitignored raw campaign data under `.context/benchmark-results`.
