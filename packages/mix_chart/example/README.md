# mix_chart gallery

A responsive gallery for the Mix-owned `mix_chart` API.

## Screens

- **Line** — curves, area fills, per-series overrides, steps, gaps, widget axis
  labels, selection, custom tooltips, viewport transforms, and Mix animation.
- **Bar** — grouped, stacked, floating, gradient, tracked, labeled, and selected
  bars.
- **Pie** — pies, donuts, labels, badges, rounded slices, selection, tooltips,
  and safe empty data.
- **Playground** — live generated-Styler controls for all three chart families.
- **Dashboard** — a polished responsive composition using the same public API.

Most examples lead with direct Flutter values. The theme switcher is a
secondary demonstration of consumer-owned Mix tokens; `mix_chart` defines no
token registry of its own.

Run from this directory:

```bash
flutter run
```
